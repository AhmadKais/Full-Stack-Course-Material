
شرح وثيقة HTML (بالعربية)
=========================

هذه الصفحة تمثل مثالًا متكاملًا على موقع ويب بلغة HTML5. فيما يلي شرح تفصيلي لكل وسم (Tag) وخصائصه.

----------------------------------
1. <!DOCTYPE html>
----------------------------------
يعلن أن الوثيقة مكتوبة بلغة HTML5.

----------------------------------
2. <html lang="en">
----------------------------------
هو العنصر الجذر لكل صفحة HTML.
الخاصية lang="en" تحدد أن اللغة الأساسية هي الإنجليزية.

مثال:
<html lang="ar">

----------------------------------
3. <head> ... </head>
----------------------------------
يحتوي على بيانات تعريفية (Meta) وروابط إلى ملفات خارجية مثل CSS و JavaScript.

مثال:
<head>
  <meta charset="UTF-8" />
  <title>أحمد قيس - صفحة تعلم HTML</title>
</head>

الوسوم المهمة:
- <meta>: لوصف محتوى الصفحة، الكلمات المفتاحية، المؤلف.
- <link>: لربط ملفات التنسيق (CSS).
- <script>: لإضافة أكواد JavaScript.

----------------------------------
4. <body> ... </body>
----------------------------------
يحتوي على جميع العناصر المرئية للمستخدم.

مثال:
<body>
  <h1>مرحبا!</h1>
</body>

----------------------------------
5. <header>
----------------------------------
يستخدم لوضع العنوان الرئيسي وروابط التنقل.

مثال:
<header>
  <h1>أحمد قيس</h1>
  <nav>
    <a href="#about">من أنا</a> | <a href="#contact">اتصل بي</a>
  </nav>
</header>

----------------------------------
6. <nav>
----------------------------------
يُستخدم لتجميع روابط التنقل بين أقسام الصفحة.

مثال:
<nav>
  <a href="#about">من أنا</a> | <a href="#projects">مشاريعي</a>
</nav>

----------------------------------
7. <main>
----------------------------------
يحتوي على المحتوى الأساسي للصفحة، ويُقسّم إلى <section>.

----------------------------------
8. <section>
----------------------------------
يستخدم لتقسيم الصفحة إلى أقسام مثل: نبذة عني، التعليم، المهارات.

مثال:
<section id="about">
  <h2>نبذة عني</h2>
  <p>أنا أحمد قيس.</p>
</section>

----------------------------------
9. وسوم تنسيق النصوص
----------------------------------
<b>، <i>، <u>، <em>، <strong>، <mark> لتغيير مظهر النص.

مثال:
<p>أنا <b>أحب</b> <em>HTML</em>!</p>

----------------------------------
10. <figure> و <figcaption>
----------------------------------
لعرض الصور أو الكود مع وصف توضيحي.

مثال:
<figure>
  <img src="photo.jpg" alt="أنا" width="200" />
  <figcaption>أنا في مكاني المفضل!</figcaption>
</figure>

----------------------------------
11. <blockquote>، <abbr>، <kbd>، <code>، <pre>
----------------------------------
للاقتباسات والاختصارات والتعليمات البرمجية.

مثال:
<blockquote>"افعل ما تحب لتنجح في عملك."</blockquote>
<abbr title="HyperText Markup Language">HTML</abbr>
<kbd>Ctrl</kbd> + <kbd>S</kbd>
<pre><code>for (int i=0; i<5; i++) { printf("مرحباً!"); }</code></pre>

----------------------------------
12. <details> و <summary>
----------------------------------
لإنشاء أقسام يمكن طيّها وفتحها (قابلة للعرض والإخفاء).

مثال:
<details>
  <summary>اضغط لعرض التفاصيل</summary>
  <p>هذه التفاصيل كانت مخفية.</p>
</details>

----------------------------------
13. القوائم
----------------------------------
<ul> قائمة غير مرتبة  
<ol> قائمة مرتبة  
<dl> قائمة تعريفية  

مثال:
<dl>
  <dt>HTML</dt>
  <dd>لغة لبناء صفحات الويب</dd>
</dl>

----------------------------------
14. الجداول <table>
----------------------------------
لعرض البيانات بشكل منظم.

مثال:
<table border="1">
  <thead><tr><th>اليوم</th><th>النشاط</th></tr></thead>
  <tbody><tr><td>الأحد</td><td>دراسة HTML</td></tr></tbody>
  <tfoot><tr><td colspan="2">نهاية الأسبوع!</td></tr></tfoot>
</table>

----------------------------------
15. <time>
----------------------------------
يُستخدم لتمثيل وقت أو تاريخ محدد.

مثال:
<time datetime="2025-10-13">13 أكتوبر 2025</time>

----------------------------------
16. <form>
----------------------------------
يُستخدم لجمع بيانات من المستخدم.

يشمل:
<input>، <textarea>، <label>، <fieldset>، <legend>

مثال:
<form action="#" method="post">
  <label for="name">الاسم:</label>
  <input type="text" id="name" name="name" required />
  <input type="submit" value="إرسال" />
</form>

----------------------------------
17. الوسائط المتعددة
----------------------------------
<img>، <audio>، <video>، <iframe>، <embed>، <object>

مثال:
<video controls>
  <source src="myvideo.mp4" type="video/mp4" />
</video>

----------------------------------
18. <aside>
----------------------------------
للمحتوى الجانبي أو الروابط السريعة.

مثال:
<aside>
  <ul>
    <li><a href="#about">من أنا</a></li>
  </ul>
</aside>

----------------------------------
19. <footer>
----------------------------------
يحتوي على معلومات ختامية مثل حقوق النشر.

مثال:
<footer>
  <p>تم الإنشاء بواسطة <i>أحمد قيس</i> &copy; 2025</p>
</footer>

----------------------------------
20. <address>
----------------------------------
لعرض معلومات التواصل أو البريد الإلكتروني.

مثال:
<address>
  يمكن مراسلتي على <a href="mailto:example@mail.com">example@mail.com</a>
</address>

----------------------------------
21. <progress> و <meter>
----------------------------------
لعرض التقدم أو النسبة المئوية.

مثال:
<progress value="70" max="100"></progress>
<meter value="0.8">80%</meter>

----------------------------------
22. <hr>
----------------------------------
يضيف خط أفقي للفصل بين الأقسام.

----------------------------------
الخلاصة:
هذه الصفحة الآن تشرح كل وسوم HTML5 المهمة المستخدمة في المثال، مع أمثلة عملية واضحة.
